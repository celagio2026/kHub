2Q Lab Exercise 1: Cafe Website

Objective:

  • 1) Demonstrate mastery in CSS and HTML by creating a Yassified SlayHub.

  • 2) Apply design concepts to produce a quality website that is able to keep up with modern design trends.

  • 3) Deliver a quality website that is responsive and is able to adapt to different screens.



Procedure:

  • 1) Work by groups of 5

  • 2) Yassify and create a Redesigned SlayHub.

  • 3) Upload the website to Yashub Pages.

  • 4) Submit the link in the Slayhub submission bin.
